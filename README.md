Description
This Java program demonstrates the use of **methods** in performing various computational tasks.  
It includes five main operations such as arithmetic calculations, factorial computation, array summarization, and prime number checking.

---

#Implemented Tasks

1. Task 1:Greeting Message  
   - Displays a welcome message to the user.  
2. Task 2: Basic Arithmetic Calculator  
   - Accepts two numbers and performs addition, subtraction, multiplication, and division.  
3. Task 3: Factorial Calculator  
   - Computes the factorial of a given number.  
4. Task 4:Array Summary  
   - Calculates and displays the sum of the array elements.  
5. Task 5: Prime Number Checker  
   - Checks whether an input number is prime.
